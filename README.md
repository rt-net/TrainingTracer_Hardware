# TrainingTracer_Hardware

[トレーニングトレーサー](https://rt-net.jp/products/rt-tracer/)
のハードウェア資料集です

![tracer](https://rt-net.jp/wp-content/uploads/2020/05/RT-Tracer.png)

## フォルダ構成

* 3d_cad_data
　トレーニングトレーサーの3DCADデータ
* circuits
　トレーニングトレーサーの回路図とNucleoの追加工マニュアル
